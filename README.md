# tf2-easycrafter
Making my own crafting interface because TF2's is clunky

<<This software is WIP/Experimental>>

# Installation/How-To:
0. Install NPM:
    - `sudo apt install npm` or equivalent for your system.
1. Clone/Download this repo and `cd` into it:
    - `cd ~/Downloads/tf2-easycrafter` (or wherever you put it)
2. Run `npm install` to install the dependencies.
3. Start the program with `npm run start` (or `npm run cli`)
    - Starts the cli interface. Type 'help' for a list of commands.
